# Project vision
Create a web app which is a game to play with your friends, called "partyW"

# Game specification
- Multiple teams play in the game, each team is two players
- Each player have paper and pen
### Game plan
- The main purpose is simple. Click on the generate button, the generate button then runs a function which randomizes one single word which is shown on the screen.
- When the word is shown, a timer starts and each team/player starts to brainstorm and write down words on paper which is relatable to the shown word. 
- When the timer is done. Each player tell their team partned what word they wrote down. For each word both players in same team wrote, the team gets one point.
- Team with most points at end of game wins
